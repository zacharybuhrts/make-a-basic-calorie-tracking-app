# PlateLog Calorie Tracker

PlateLog is a small browser-based calorie tracking app for logging foods, meal types, dates, and daily calorie progress.

## Features

- Add foods with calories, meal type, and date
- See today's total calories, remaining calories, and goal progress
- Update the daily calorie goal
- Filter the food log between today and all entries
- Delete logged foods
- Save data locally in the browser

## Run it

Open `index.html` in any modern browser.

No install or build step is required.
